# 🛡️ DeepFakeShield
AI-Powered Deepfake Detection & Media Authenticity Verification System
## ABOUT PROJECT

DeepFakeShield is an AI-powered web application designed to detect deepfake images and videos using advanced deep learning techniques.

With the rapid growth of AI-generated synthetic media, misinformation and identity misuse are increasing. DeepFakeShield aims to provide a reliable and easy-to-use platform for verifying the authenticity of digital media.

The system analyzes uploaded media files and determines whether they are Real or Fake, helping users combat misinformation and digital fraud.
## 🚀 TECH STACK
DeepFakeShield is built using a modern full-stack architecture that combines web technologies with deep learning frameworks for accurate media authenticity detection.

👩‍💻 Frontend
The frontend is designed to provide a clean, responsive, and user-friendly interface for uploading media and viewing results.

🔹 HTML5

Structures the web pages and UI components

Handles forms for image/video uploads

Ensures semantic and accessible markup

🔹 CSS3

Styles the application layout

Provides responsive design for different screen sizes

Enhances UI/UX with animations and transitions

🔹 JavaScript

Handles client-side validation

Manages file upload interactions

Dynamically updates prediction results

Improves overall user interactivity

🧠 Backend
The backend processes uploaded files, runs AI predictions, and returns classification results securely.

🔹 Python

Core programming language for backend logic

Handles server-side processing

Integrates AI/ML model with web framework

🔹 Flask

Lightweight Python web framework

Manages API routes (e.g., /predict)

Handles HTTP requests and responses

Manages file uploads securely

Connects frontend with AI model
🤖 AI / ML
The AI system is responsible for detecting deepfake patterns in uploaded media.

🔹 TensorFlow

Deep learning framework used to build and train the model

Provides efficient model execution

🔹 Keras

High-level API for building neural networks

Simplifies CNN model development

Enables faster experimentation and training

🔹 OpenCV

Image and video preprocessing

Frame extraction from videos

Face detection and normalization

Resizing and filtering before model input

🔹 Convolutional Neural Network (CNN)

Learns spatial features from images

Detects facial inconsistencies and manipulation artifacts

Outputs probability score for Real vs Fake classification
